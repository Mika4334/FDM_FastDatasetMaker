# FastDatasetMaker-YT-
Remove vocals from YouTube video
I'm noob at this

# Info
create venv ```python -m venv```

install some reqs ```pip install requirements.txt```

open ```FastDatasetMaker.py```

enter any yt link FOR EXAMPLE: ```https://www.youtube.com/watch?v=rQT3kytoUrg```

enjoy :)
